Binary Tree v1 (Ocampo)

This is a basic binary tree visualizer used in Rika Yoshii's CS311 Class.
Each vertex in a binary tree consists of 3 elements.

struct vertex {
vertex * left; // Pointer to the Left Vertex
int elem; // Integer Id
vertex * right; // Pointer to the Right Vertex
};

In addition, there are three vertex pointers to account for: root, parent, and v. These are represented by the colored circles around each node.

vertex * root, parent, v;

Instructions: Make the parent pointer and v pointer go down the tree until the V pointer becomes NULL and the Parent pointer points to a leaf.
